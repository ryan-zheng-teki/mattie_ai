Problem Statement

In a rectangular coordinate system xOy, there are two ellipses:

Ellipse C: x²/4 + y² = 1

Ellipse E: x²/16 + y²/4 = 1

Point P is an arbitrary point on ellipse C. The ray OP (from origin O) intersects ellipse E at point Q.

Find the value of the ratio |OQ|/|OP|.

Solution Steps

  • 1. Draw the coordinate system and ellipses

    Draw the coordinate axes and both ellipses C and E.

  • 2. Select point P on ellipse C

    Choose any point P on ellipse C and mark it.

  • 3. Draw ray OP and find intersection Q

    Draw ray OP and find where it intersects ellipse E at point Q.

  • 4. Calculate ratio |OQ|/|OP|

    Calculate and display the ratio of distances |OQ|/|OP|.

  • 5. Demonstrate constant ratio

    Show that this ratio remains constant (= 2) regardless of P's position.

  • Clear & Reset

    Clear the canvas and restart exploration.

Ratio |OQ|/|OP| = 0
