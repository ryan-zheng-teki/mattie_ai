问题描述

在直角坐标系 xOy 中，有两个椭圆：

椭圆 C: x²/4 + y² = 1

椭圆 E: x²/16 + y²/4 = 1

点 P 是椭圆 C 上的任意点。从原点 O 出发的射线 OP 与椭圆 E 相交于点 Q。

求比值 |OQ|/|OP| 的值。

解题步骤

  • 1. 绘制坐标系和椭圆

    绘制坐标轴和两个椭圆 C 和 E。

  • 2. 选择椭圆 C 上的点 P

    在椭圆 C 上选择任意点 P 并标记。

  • 3. 绘制射线 OP 并找出交点 Q

    绘制射线 OP 并找出其与椭圆 E 的交点 Q。

  • 4. 计算比值 |OQ|/|OP|

    计算并显示距离比值 |OQ|/|OP|。

  • 5. 证明比值恒定

    证明无论点 P 在椭圆 C 上的位置如何，此比值始终为 2。

  • 清除并重置

    清除画布并重新开始探索。

比值 |OQ|/|OP| = 0
